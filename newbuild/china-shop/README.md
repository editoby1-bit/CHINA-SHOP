# China Shop Frontend

China Shop is a full frontend-only, hash-routed e-commerce web app built with vanilla HTML, CSS, and JavaScript.

## Files

- `index.html`
- `css/style.css`
- `js/data.js`
- `js/router.js`
- `js/app.js`

## How to run locally

1. Extract the ZIP.
2. Open the project folder.
3. Double-click `index.html` to open it in your browser.
   - Or right-click `index.html` and choose **Open with Chrome** or any modern browser.
4. Since the app uses hash-based routing and localStorage, it works locally without a backend.

## Features

- Buyer-facing marketplace with product, shop, cart, checkout, and special request pages
- Hash-based SPA routing with no page reloads
- Responsive layout for desktop, tablet, and mobile
- localStorage-powered seed data, cart, products, orders, and request queue
- Vendor dashboard for product management and order status updates
- Admin dashboard for revenue, commission, order overview, request queue, and vendor list

## Notes

- Images use `https://picsum.photos` placeholders.
- All prices are displayed in Nigerian Naira.
- Buyer-facing views do not reveal sourcing details.
